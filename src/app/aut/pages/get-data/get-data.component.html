Conclusion

When using Angular Universal, the components will be initialized twice by server and browser. And the server calls Angular LifeCycle hooks from `ngOnInit()` to `ngOnDestroy()`.

To prevent it, Using `ServerTransferStateModule` and `BrowserTransferStateModule`. It's similar with `LocalStorage` but can share the data between server and browser platforms.
